# Number Cruncher
## Objectif
Développer une calculatrice simple permettant d’effectuer les opérations mathématiques de base
## Installation
### Prérequis
- `Git` installé localement
- `Python` et `pip` installé
### Instructions
1. Cloner le projet localement à l'aide du terminal:
```
git clone https://github.com/ztaous/TP3_LOG3000.git
```
2. Aller dans la racine du projet et créer son environnement virtuel:
```
cd TP3_LOG3000
python -m venv venv
```
3. Activer l'environnement virtuel:
```
./venv/bin/activate ## Linux
./venv/Scripts/activate ## Windows
```
4. Installer flask:
```
pip install flask
```
6. Éxecuter l'application sur `localhost:5000`:
```
cd src
flask run
```
